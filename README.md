# XML to JSON translator project
## This program knows how to translate XML to JSON
## Accepted structures:
* Object with value:   
``` <objectName> value </objectName> ```
* Object with subobjects:   
``` 
    <objectName>  
        <suobjectName1>...</suobjectName1>  
        <suobjectName2>...</suobjectName2>  
    </objectName>
```
